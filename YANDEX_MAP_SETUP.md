# Настройка Яндекс карт в контактах

## Текущая настройка

Карта настроена для отображения адреса офиса. Вы можете легко изменить адрес и координаты.

## Как изменить адрес

Откройте файл `src/components/contacts-modal.tsx` и найдите константы в начале файла:

```typescript
const APP_ADDRESS = "г. Симферополь, ул. Примерная, д. 1, офис 100";
```

Замените на ваш реальный адрес.

## Как указать точные координаты (рекомендуется)

Для более точного отображения карты рекомендуется указать координаты вручную:

1. Найдите координаты вашего адреса:
   - Откройте [Яндекс.Карты](https://yandex.ru/maps)
   - Найдите ваш адрес
   - Кликните правой кнопкой мыши на нужное место
   - Выберите "Что здесь?"
   - Скопируйте координаты (формат: долгота, широта)

2. Обновите константу `OFFICE_COORDINATES` в файле `src/components/contacts-modal.tsx`:

```typescript
// Пример для Симферополя
const OFFICE_COORDINATES: [number, number] | undefined = [34.1003, 44.9482];
```

**Важно:** Координаты указываются в формате `[долгота, широта]` (не наоборот!)

3. Если координаты не указаны (`undefined`), карта автоматически определит их по адресу через геокодирование.

## Примеры координат для популярных городов

- **Москва (Красная площадь)**: `[37.6173, 55.7558]`
- **Санкт-Петербург (Эрмитаж)**: `[30.3141, 59.9386]`
- **Симферополь (центр)**: `[34.1003, 44.9482]`
- **Казань (центр)**: `[49.1064, 55.8304]`

## Настройка масштаба карты

Вы можете изменить масштаб карты, изменив параметр `zoom` в компоненте:

```typescript
<YandexMap
  address={APP_ADDRESS}
  center={OFFICE_COORDINATES}
  zoom={16}  // Измените значение (от 1 до 19, где больше = ближе)
  height="400px"
/>
```

Рекомендуемые значения:
- `10-12` - для показа города/района
- `14-16` - для показа улицы (рекомендуется)
- `17-19` - для показа конкретного здания

## Высота карты

Вы можете изменить высоту карты, изменив параметр `height`:

```typescript
<YandexMap
  address={APP_ADDRESS}
  center={OFFICE_COORDINATES}
  zoom={16}
  height="500px"  // Измените на нужную высоту
/>
```

## API ключ Яндекс карт (опционально)

По умолчанию карта работает без API ключа, но для продакшена рекомендуется получить API ключ:

1. Перейдите на [Яндекс.Кабинет разработчика](https://developer.tech.yandex.ru/)
2. Создайте новый ключ для JavaScript API и HTTP Геокодер
3. Обновите URL загрузки API в файле `src/components/yandex-map.tsx`:

```typescript
script.src = "https://api-maps.yandex.ru/2.1/?apikey=ВАШ_API_КЛЮЧ&lang=ru_RU";
```

**Примечание:** Без API ключа есть ограничения на количество запросов, но для большинства сайтов этого достаточно.

## Проверка работы

После изменения адреса или координат:

1. Сохраните файл
2. Перезапустите dev сервер (если он запущен)
3. Откройте страницу контактов
4. Проверьте, что карта отображается правильно и метка находится на нужном месте

## Решение проблем

### Карта не загружается
- Проверьте подключение к интернету
- Убедитесь, что нет блокировщиков рекламы, которые могут блокировать Яндекс.Карты
- Проверьте консоль браузера на наличие ошибок

### Неправильное местоположение
- Убедитесь, что координаты указаны в правильном формате `[долгота, широта]`
- Проверьте, что адрес написан правильно и полностью
- Попробуйте указать координаты вручную вместо автоматического геокодирования

### Метка не отображается
- Проверьте, что координаты находятся в допустимом диапазоне
- Убедитесь, что адрес не пустой
